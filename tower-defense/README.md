# Tower Defense

![Tower Defense Game Preview](preview.png)

A modular Tower Defense game built using Python 3.13 and Pygame.The game features 5 distinct tower classes, enemy waves with progressive difficulty, a robust upgrade system, and persistent save/load functionality.

---

## Features
- 5 Tower Types (Basic, Sniper, Splash, Slow, Rapid)
- 5 Enemy Types (Normal, Fast, Tank, Healer, Boss)
- Tower Upgrades (Level 1-3 with stat scaling)
- Advanced Targeting Priorities (First, Strong, Weak, Close)
- Save and Load game state (JSON)
- Game Speed Control (1x / 2x toggle)
- Floating Text for Damage and Rewards
- Interactive Sidebar with Tower Metadata (Cost, Damage)
- 25 Waves of progressive difficulty

---

## Controls
- Left Click — Select Tower / Place Tower / Upgrade / Sell
- Right Click — Deselect Tower
- Keys 1-5 — Quick Select Tower Types
- SPACE — Start Next Wave
- S — Quick Save Game
- ESC — Deselect Current Selection
